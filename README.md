# Digital Store

Интернет-магазин цифровых товаров на Next.js + TypeScript.

## Стек
- **Frontend:** Next.js 14, TypeScript, Tailwind CSS
- **Backend:** Next.js API Routes
- **База данных:** PostgreSQL + Prisma ORM
- **Аутентификация:** NextAuth.js
- **Оплата:** Stripe
